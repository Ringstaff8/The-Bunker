# The Bunker

> A Google Apps Script Inventory Management & Point of Sale System for School Stores

The Bunker is a web-based Inventory Management and Point of Sale (POS) application built with Google Apps Script and Google Sheets. It was developed to replace spreadsheet-based inventory tracking with a structured, easy-to-use system for managing products, sales, promotional giveaways, and physical inventory.

Originally designed for the **Paducah Middle School** school store, the project demonstrates how Google Workspace can be used to build a full-featured inventory system without requiring a traditional database or paid software.

---

## Features

### Inventory Management

- Product catalog management
- SKU management
- Categories, Collections, and Designs
- Size tracking
- Vendor information
- Reorder levels
- Inventory valuation
- Active/Inactive products

### Point of Sale

- Shopping cart
- Cash and card sales
- Automatic inventory deduction
- Profit calculations
- Transaction history
- Sales logging

### Promotional Items

- Student giveaways
- Promotional inventory tracking
- Reason tracking
- Notes
- Inventory deduction
- Promotional transaction history

### Physical Inventory

- Inventory Sessions
- Resume saved inventory counts
- Category-by-category counting
- Session progress tracking
- Physical inventory application
- Inventory adjustment transactions
- Complete inventory audit trail

### Dashboard

- Inventory statistics
- Inventory value
- Low inventory monitoring
- Daily sales
- Promotional tracking

# Development Philosophy

The Bunker is developed using an incremental approach.

Every feature follows these principles:

- One feature at a time
- One change per test
- Reuse existing helper functions
- Maintain backward compatibility whenever possible
- Commit each completed feature to Git

---

# Why This Project?

Commercial inventory systems are often expensive and overly complex for small school stores.

The Bunker was created to provide a free, maintainable alternative that runs entirely within Google Workspace while still offering features commonly found in commercial inventory software.
